# Project Scope

## v0 In Scope

- Single workstation target: RTX 5090 class GPU first.
- Canary Islands 3 km before 1 km.
- One domain path before nested generality.
- One initial physics suite.
- One validated dycore path or reduced dycore milestone.
- WRF fixture extraction and tolerance metadata.
- Profiler-first GPU acceptance.

## v0 Out Of Scope

- Full WRF option matrix.
- General global model support.
- Production multi-GPU scaling.
- Every WRF physics suite.
- Data assimilation.
- ML emulator coupling except as research notes.
- Public branding as WRF.

## Expansion Rule

Scope expansion requires a manager proposal, reviewer critique, risk update, and explicit human approval.
