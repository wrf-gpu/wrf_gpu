# Glossary

- WRF-compatible: compatible where useful in variables, fixtures, namelist mapping, and validation behavior.
- Physics-identical-enough: statistically and physically consistent with trusted WRF or analytic references under documented tolerances.
- Whole-state GPU residency: high-frequency model state remains on the GPU through timestep loops.
- Proof object: a test result, profiler artifact, ADR, fixture comparison, or reproducible command that supports a claim.
