# Project Facts

- Project name: `wrf_gpu2`.
- Purpose: GPU-native, WRF-compatible regional NWP system from scratch.
- First target: Canary Islands 3 km then 1 km daily runs.
- Hardware target: single RTX 5090 32 GB class GPU first.
- Backend is not selected until M2.
- Old global `wrf-gpu-port` skill is not applicable to this project.
