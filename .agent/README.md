# AgentOS

This directory is the project-local operating system for agents. It is intentionally committed to git so managers, workers, reviewers, and testers share the same rules.

Authoritative order:

1. `PROJECT_CONSTITUTION.md`
2. `AGENTS.md`
3. current sprint contract
4. relevant `.agent/skills/<name>/SKILL.md`
5. on-demand references

Do not load or follow the old global `wrf-gpu-port` skill for this repository. It belongs to a separate failed legacy-port effort.
