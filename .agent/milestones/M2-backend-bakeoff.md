# M2 - Backend Bakeoff

Goal: choose the primary backend with evidence, not preference.

Deliverables:

- representative stencil in candidate backends
- representative physics column kernel in candidate backends
- correctness report
- profiler JSON
- ADR selecting the initial stack

Acceptance gates:

- same fixture and metrics across candidates
- reviewer challenges maintainability and agent success rate
- human approval if decision is hard to reverse
