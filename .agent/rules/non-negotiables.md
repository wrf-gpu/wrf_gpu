# Non-Negotiables

- No model implementation without a sprint contract.
- No merge without independent tests.
- No physics claim without WRF fixture, analytic oracle, or invariant evidence.
- No GPU performance claim without profiler artifact.
- No host/device transfer inside timestep loops without explicit approval.
- No architecture change without ADR.
- No memory, rule, or skill update without patch, evidence, review, and validation.
- No parallel edits to shared core files.
